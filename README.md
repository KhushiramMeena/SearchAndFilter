# Frontend Assignment

## Installation Process

1. **Clone the Repository:**
    ```bash
    git clone https://github.com/KhushiramMeena/SearchAndFilter.git
    ```

2. **Navigate to the Project Directory:**
    ```bash
    cd SearchAndFilter
    ```

3. **Install Dependencies:**
    ```bash
    npm install
    ```

4. **Start the project**
    ```bash
    npm run dev
    ```
